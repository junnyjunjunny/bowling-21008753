# Plan: 스트라이크가 있는 경우 보너스 점수 계산

## 목표 (RED)

한 프레임에서 스트라이크(첫 투구로 10핀)가 발생했을 때, 해당 프레임이 투구 1회로 종료되고 다음 두 번의 투구가 보너스로 더해지는지 검증하는 테스트를 작성한다.

## 작성할 테스트

파일: `tests/test_game.py`

```python
def test_one_strike_adds_next_two_rolls_as_bonus():
    game = Game()

    game.roll(10)  # 1프레임: 스트라이크
    game.roll(3)   # 2프레임 첫 투구
    game.roll(4)   # 2프레임 두 번째 투구

    for _ in range(16):  # 3~10프레임: 전부 거터
        game.roll(0)

    assert game.score() == 24
```

- 프레임별 기대 점수: 1프레임 = 10 + 3 + 4(보너스) = 17, 2프레임 = 3 + 4 = 7, 3~10프레임 = 0. 합계 24.
- 이번 투구는 총 1(스트라이크) + 2(2프레임) + 16(3~10프레임) = 19회.
- 현재 구현은 프레임마다 항상 2개의 투구를 소비한다고 가정하므로, 스트라이크 이후 `rolls` 인덱스가 어긋나 잘못된 값을 계산할 것으로 예상된다 (실패 예상, 정확한 오답 값은 실행해서 직접 확인). → 진짜 RED.
